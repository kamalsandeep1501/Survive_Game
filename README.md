# 🎮 Survive Game

A simple survival-style mobile game built using **Flutter** and **Dart**.  
In this game, the player must protect the base from falling stones while earning points by tapping and destroying them.

---

# 🌐 Live Demo

You can play the game directly in your browser.

🎮 **Play the Game Online**

👉 https://tapsurvivor-88a47.web.app/

This version of the game is deployed using **Firebase Hosting** and works on desktop and mobile browsers.

---

# 📱 Game Overview

- A **base** is located at the bottom of the screen.
- **Red stones fall from the sky** continuously.
- If a stone **hits the base**, the **base health decreases**.
- If the player **taps the stone**, it gets destroyed and the **score increases**.
- The objective is to **survive as long as possible** and achieve the **highest score**.

# 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/kamalsandeep1501/Survive_Game.git
```

### 2️⃣ Navigate to the project folder

```bash
cd Survive_Game
```

### 3️⃣ Install dependencies

```bash
flutter pub get
```

### 4️⃣ Run the application

```bash
flutter run
```

Make sure **Flutter SDK** is installed and a device/emulator is connected.
